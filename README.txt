This project uses:

# JSP 2.0
# Struts for MVC
    * index.jsp as the only view
    * Model classes Extract.java and Insert.java for basic insertion and extraction operations
    * Thin layer around Extract.java called Home.java
    * Sessions thourgh cookies
# XML for storing and accessing data
# HTML5 for frontend

Files to watch:
# web/index.jsp (View)
# web/*.xml (Data storage files)
# web/WEB-INF/struts-config.xml (Supposed to be the controller)
# src/java/lolcat.properties for static values used in the views
# src/java/cat folder for Insert.java, Extract.java, Home.java
#
